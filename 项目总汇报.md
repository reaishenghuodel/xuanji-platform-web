## 一、项目概况

### 1.1 项目定位

**璇玑智科人工智能一体化应用平台**（以下简称「璇玑智科平台」）是一款面向品牌营销团队的 **GEO（Generative Engine Optimization，生成式引擎优化）智能运营 SaaS 平台**。

**核心理念**：在 AI 大模型时代，用户不再仅通过传统搜索引擎获取信息，而是直接向 AI（DeepSeek、ChatGPT、文心一言等）提问。品牌需要一套全新方法论，确保在 AI 对话场景中被准确理解、优先推荐。GEO 正是这一新赛道的核心能力。

### 1.2 对标平台

| 对标平台 | 类型 | 参考内容 |
|---------|------|----------|
| [geo.mxspark.com](https://geo.mxspark.com) | 国内 GEO 监控平台 | 品牌 AI 监测、诊断报告、可视化数据看板 |
| [dso.aidso.com](https://dso.aidso.com) | 极智星智能选词 | 智能选词、AI 拓词、内容创作、达人素材库 |

### 1.3 技术架构

| 层级 | 技术选型 | 说明 |
|------|---------|------|
| 框架 | React 19 + TypeScript | 最新稳定版本，严格类型检查 |
| 构建工具 | Vite 8 | 快速 HMR，优化构建产物 |
| 样式 | Tailwind CSS 4 | 原子化 CSS，暗色主题 |
| 图表 | Recharts 3 | 丰富的数据可视化能力 |
| 图标 | Lucide React | 统一、轻量的 SVG 图标 |
| 路由 | React Router DOM 7 | HashRouter，支持静态部署 |
| 代码规范 | Oxlint + TypeScript | 零错误构建标准 |

### 1.4 设计风格

以 **「深空科技」** 为主题，灵感来源于「璇玑」（古代天文仪器）的天文学意象：

- **主背景**：`#0B1120`（深空蓝黑）
- **卡片面**：`#1E293B`（太空灰蓝）
- **强调色**：`#0EA5E9`（天蓝）、`#2DD4BF`（极光青）、`#6366F1`（星紫）
- **成功/警告**：`#10B981` / `#F59E0B`
